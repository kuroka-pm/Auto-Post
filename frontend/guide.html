🚀 はじめに

AutoPost v2 は、X (Twitter) と Threads への投稿を AI でサポートする自動投稿ツールです。 トレンド分析、ペルソナ設定、AI による文章生成、スケジュール投稿などの機能を搭載しています。

⚙️ 初期設定

Step 1: API キーの設定

上部ナビバーの「⚙️ 設定」→「🔑 API設定」タブで各サービスの API キーを入力してください。

  • Gemini API Key — Google AI Studio で取得
  • X (Twitter) API — X Developer Portal で取得（API Key, Secret, Access Token, Access Token Secret の4つ）
  • Threads API — Meta Developer Portal で取得したアクセストークン

設定後、「🔌 接続テスト」ボタンで接続を確認してください。

Step 2: ペルソナの設定

「👤 ペルソナ」タブで投稿者の人物像を設定します。年齢・職業・趣味・性格などを入力し、 「🧠 AI生成」ボタンで詳細なペルソナテキストを自動生成できます。

Step 3: プロンプトの設定

「📝 プロンプト」タブで書き方の方針やNG表現を設定します。 ここで設定したルールが AI の投稿生成時に適用されます。

Step 4: 情報源の設定

トレンド分析ページの「📡 ソース設定」パネルでトレンド取得用 RSS ソースを管理します。 「🔑 ペルソナからキーワード提案」ボタンをクリックすると、ペルソナに合ったキーワードが提案されます。 選択して「✅ 登録」すると、自動で Google News RSS URL が生成・保存され、トレンドも自動更新されます。

  • 登録済みソースはテーブル形式（☑ キーワード | RSS URL）で一覧表示
  • チェックボックスで選択 →「🗑️ 選択削除」で個別削除
  • 「🗑️ 全削除」ですべてのソースをリセット

📝 主な機能

ダッシュボード

投稿数、インプレッション、いいね数などの KPI を確認できます。 エンゲージメントデータは X アナリティクスの CSV をインポートすることで更新されます。

トレンド分析

登録した RSS ソースから最新トレンドを取得します。 各トレンドにはソース元の URL がリンク表示され、不要なトレンドは個別に削除できます。 「🧠 AIおすすめ」ではペルソナとの相性が良いトレンドを AI が分析・提案します。 トレンドリストとソース設定は折りたたみパネルで表示され、更新時に自動で展開されます。

手動投稿（AI 投稿ジェネレーター）

トレンドとスタイルを選択し、AI が投稿文を自動生成します。 生成された投稿は編集後、各投稿ごとに投稿先（X / Threads）を個別選択してワンクリックで投稿できます。 画像を添付し、ALT テキストを設定することも可能です。

利用可能なライティングスタイル（8種）:

  • 一撃キャッチコピー — 80-140文字。結論ファーストの一言凝縮
  • ニュース×一言 — 80-140文字。ニュース引用＋短い私見
  • 共感エピソード — 140-250文字。過去の体験をトレンドに接続
  • 構造分析 — 140-280文字。因果関係を淡々と解説
  • 概念の再定義 — 140-250文字。独自の新しい定義を提示
  • 有益リスト型 🆕 — 200-280文字。まとめ・Tips形式。保存＆RTされやすい
  • 逆説インサイト型 🆕 — 180-260文字。否定連鎖→真の答えパターン
  • 問いかけ共感型 🆕 — 120-200文字。あるある描写でリプライ誘発

スケジューラ

曜日ごとの投稿時間を設定し、自動投稿をスケジュールできます。 ランダムジッター機能により、機械的な投稿タイミングを避けます。 「▶ 開始」でスケジューラが起動し、設定した時間に自動で投稿を生成・送信します。

アナリティクス

X アナリティクスからダウンロードした CSV ファイルをインポートして、 エンゲージメントの推移をグラフで視覚的に確認できます。 AI 分析機能で投稿の傾向分析や改善アドバイスも取得可能です。

note 連携

note.com のマイページ URL を登録すると、公開記事を取得して一覧表示します。 取得した記事データはキャッシュされるため、再起動後も表示が維持されます。

📱 レスポンシブデザイン

画面幅に応じ3段階のブレークポイントで自動的にレイアウトが調整されます。

  • ≤1024px（タブレット）— KPIと時間グリッドが2列に
  • ≤768px（スマホ横）— ナビバーがコンパクトに折りたたみ
  • ≤480px（スマホ縦）— 全グリッドが1列表示

💡 Tips

設定のバックアップ

設定ページの「📤 エクスポート」ボタンで現在の設定を JSON ファイルに保存できます。 「📥 インポート」で復元、「🔄 初期化」でデフォルト設定にリセットできます。

🌗 ダーク / ライトモード

ナビバー右端のテーマボタンでダークモードとライトモードを切り替えできます。 選択したテーマはブラウザに保存され、次回起動時に復元されます。

🚀 バズ最適化ヒント

プロンプトには 2025-2026 年の X アルゴリズムに対応したバズ強化ルールが組み込まれています。 具体的な数字・メタファー・否定連鎖フック・短文リズムなどを AI が自動的に反映します。

Threads トークンの有効期限

Threads のアクセストークンは 60 日で期限切れになります。 API設定タブの「🔄 トークン更新」ボタンで長期トークンに更新してください。 有効期限はナビバーのステータスドットでも確認できます。

CSV インポート

X アナリティクスから CSV をダウンロードし、プロジェクトフォルダの INBOX フォルダに配置すると、 アナリティクスページの「📥 CSVインポート」で取り込みできます。

ページの記憶

最後に開いていたページが記憶され、次回起動時に同じページで再開します。

ログファイル

実行ログは logs/ フォルダに日付別のファイルで自動保存されます。 エラー発生時のデバッグに利用できます。

API キーの安全管理

API キーはローカルの config.json に保存されます。 このファイルを第三者に共有しないでください。 Git リポジトリにコミットする場合は .gitignore に含めてください。
